# CityPulse AI

CityPulse AI is a smart infrastructure health and anomaly monitoring system
that detects early failures in urban infrastructure using rule-based logic
and machine learning.

## Problem
Urban infrastructure failures occur without warning, causing safety risks
and high maintenance costs.

## Solution
Our system monitors sensor data, detects anomalies, assigns health scores,
and alerts authorities before failures occur.

## Tech Stack
- Python
- Machine Learning (Isolation Forest)
- Streamlit Dashboard

## How to Run
```bash
pip install -r requirements.txt
python backend/app.py
streamlit run frontend/dashboard.py
